# Password Reset App

A full-stack password reset flow built with React, Node.js, Express, and MongoDB.

## Live Demo
- Frontend: https://password-reset-frontend-3k9x.onrender.com
- Backend: https://password-reset-backend-m4n4.onrender.com

## Tech Stack
- React + Bootstrap (Frontend)
- Node.js + Express.js (Backend)
- MongoDB + Mongoose
- Nodemailer + Mailtrap
- JWT + Bcrypt

## Features
- Forgot Password (sends reset email)
- Reset Password (token verification + expiry)
- Secure password hashing
